# The-hackclub-study

A mission-control study timer with XP, levels, and a to-do workflow.
This repo includes a dashboard and a scrapyard tab for future feature expansion.

## Files
- `index.html` — main app entry point for GitHub Pages
- `rayaan.html` — original project file
- `style.css` — app styling

## Usage
Open `index.html` in your browser.

## GitHub Pages
To publish this project on GitHub Pages, use the `main` branch and deploy from the repository root.

## Notes
Complete tasks to earn XP and level up. The scrapyard section is ready for future content.
